## Project Overview
This project analyzes machine telemetry data for Daikibo, a manufacturing company, as part of the Deloitte Data Analytics Virtual Experience.

## Objective
- Identify machine downtime
- Compare downtime across factories
- Analyze machine-level failures

## Tools Used
- Tableau Public
- CSV / JSON data

## Key Insights
- Certain factories show significantly higher downtime
- Specific machine types contribute most to failures
- Interactive dashboard helps identify problem areas quickly
